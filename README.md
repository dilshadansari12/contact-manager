# Contact manager..

Contact manager is a contact saver palatform , wich have a some simple function 


## run project on local system
* Download a project and run in cmd


```bash
npm install
```


Then Run

```bash
npm start
```

## Usage
* Save a Contact 
* Call Contact
* Delete Contact
